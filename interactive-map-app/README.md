# 交互式世界地图应用

这是一个基于Flask和Leaflet的交互式世界地图应用，支持国家、省级和市级的层级浏览，以及标记功能。

## 功能特性

- 显示完整的世界地图，包含主要国家的地理边界
- 实现地图缩放、平移等基本交互操作
- 支持国家、中国省份和城市的点击交互
- 点击任意区域可查看详细信息
- 支持在地图上添加和删除标记
- 标记数据持久化存储
- 响应式设计，适配不同设备

## 技术栈

- 后端：Python Flask
- 前端：HTML, CSS, JavaScript
- 地图库：Leaflet
- 数据存储：JSON文件

## 安装和运行

1. 确保系统已安装Python 3.x
2. 安装Flask：
   ```bash
   pip install flask
   ```
3. 运行应用：
   ```bash
   python app.py
   ```
4. 在浏览器中访问 http://localhost:5000

## 使用说明

1. **浏览地图**：使用鼠标滚轮缩放，拖拽平移
2. **查看国家信息**：点击国家区域查看基本信息
3. **中国地区浏览**：点击中国区域进入省级视图，点击省份进入市级视图
4. **添加标记**：点击"添加标记"按钮，然后在地图上点击要标记的位置
5. **删除标记**：点击标记上的删除按钮
6. **导航**：使用控制面板中的按钮在不同层级间切换

## 文件结构

- `app.py`: Flask应用主文件
- `templates/index.html`: 主页面模板
- `map_data.json`: 存储标记数据的文件

## 开发说明

如需扩展功能，可以：
- 在app.py中添加更多的API端点
- 在index.html中添加更多的地图数据
- 增强前端交互体验

## 注意事项

- 应用使用本地JSON文件存储数据，适用于小型应用
- 地图数据为简化版，实际项目中可以使用更精确的地理数据
- 标记坐标使用WGS84标准
