# 图片相似度查找工具

这个程序可以帮您在一个文件夹中查找与输入图片相似的图片。它会先对输入图片进行预处理（去除白色边框并调整大小至58x58像素），然后再与其他图片进行比较。

## 功能特点

- 自动检测并移除图片的白色边框
- 将所有图片调整到统一尺寸（58x58像素）进行比较
- 使用均方误差算法计算图片相似度
- 支持常见的图片格式（JPG, JPEG, PNG, BMP, TIFF, GIF）

## 安装依赖

在使用本程序前，请确保安装了以下Python库：

```bash
pip install pillow numpy
```

## 使用方法

### 命令行使用

```bash
python find_similar_images.py <输入图片路径> <待搜索文件夹路径> [--threshold 阈值] [--top-n 返回数量]
```

### 参数说明

- `<输入图片路径>`: 您要用来搜索相似图片的原始图片路径
- `<待搜索文件夹路径>`: 包含待比较图片的文件夹路径
- `--threshold`: 相似度阈值（0-1之间，默认为0.3），只有相似度高于此值的图片才会被返回
- `--top-n`: 返回最相似的图片数量（默认为10）

### 示例

```bash
# 查找相似图片，返回最相似的5张（相似度阈值为0.5）
python find_similar_images.py input.jpg ./images --threshold 0.5 --top-n 5

# 只指定图片和文件夹（使用默认参数）
python find_similar_images.py photo.png ./my_photos
```

## 输出结果

程序会输出找到的相似图片列表，按照相似度从高到低排序，格式如下：

```
图片路径: 相似度分数
```

相似度分数范围是0到1，数值越接近1表示越相似。

## 技术细节

1. **白边移除**: 程序会自动检测图片中的白色区域，并保留主要内容部分
2. **尺寸标准化**: 所有图片都被调整到58x58像素以确保公平比较
3. **相似度算法**: 使用均方误差(MSE)来计算图片之间的相似度
