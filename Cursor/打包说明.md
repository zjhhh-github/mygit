# 售前通讯录查询工具 - 打包说明

## 快速打包

### 方法一：使用批处理脚本（推荐）

双击运行 `build_exe.bat`，脚本会自动完成以下步骤：
1. 检查并安装 PyInstaller
2. 打包程序
3. 清理临时文件

打包完成后，可执行文件位于 `dist\售前通讯录查询.exe`

---

### 方法二：手动打包

#### 1. 安装 PyInstaller
```bash
pip install pyinstaller
```

#### 2. 执行打包命令
```bash
pyinstaller --onefile --windowed --add-data "售前通讯录.txt;." --name "售前通讯录查询" db_viewer.py
```

**参数说明：**
- `--onefile`：打包成单个 exe 文件
- `--windowed`：无控制台窗口（GUI 程序）
- `--add-data "售前通讯录.txt;."`：将文本文件打包到 exe 中
  - Windows 使用分号 `;` 分隔源路径和目标路径
  - Linux/Mac 使用冒号 `:` 分隔
- `--name "售前通讯录查询"`：指定生成的 exe 文件名

#### 3. 清理临时文件（可选）
```bash
rmdir /s /q build
del "售前通讯录查询.spec"
```

## 使用说明

### 用户使用
1. 将 `售前通讯录查询.exe` 发送给用户
2. 用户双击运行即可，无需安装 Python 或其他依赖
3. `售前通讯录.txt` 已内置在 exe 中，无需额外提供

### 注意事项
- exe 文件体积约 15-20MB（包含 Python 运行时 + 依赖库 + 数据文件）
- 首次运行可能需要几秒钟解压临时文件
- 杀毒软件可能误报，需要添加信任

---

## 开发环境运行

如果需要在开发环境中运行（不打包）：

```bash
python db_viewer.py
```

程序会自动检测运行环境：
- **打包后**：从 `sys._MEIPASS` 临时目录读取 `售前通讯录.txt`
- **开发环境**：从当前目录读取 `售前通讯录.txt`

## 常见问题

### Q: 打包后 exe 无法运行？
A: 检查以下几点：
- 确保 `售前通讯录.txt` 存在于项目目录
- 确保文本文件编码为 UTF-8
- 查看是否被杀毒软件拦截

### Q: 如何减小 exe 体积？
A: 
- 使用 `--exclude-module` 排除不需要的模块
- 使用 UPX 压缩（`--upx-dir`）
- 不使用 `--onefile`，改为目录模式

### Q: 如何添加图标？
A: 
```bash
pyinstaller --onefile --windowed --add-data "售前通讯录.txt;." --icon="icon.ico" db_viewer.py
```

## 技术细节

### 资源路径获取函数
```python
def get_resource_path(relative_path: str) -> str:
    """
    获取资源文件的绝对路径（支持打包后的exe）
    """
    try:
        # PyInstaller 打包后的临时文件夹路径
        base_path = sys._MEIPASS
    except AttributeError:
        # 开发环境，使用当前文件所在目录
        base_path = os.path.dirname(os.path.abspath(__file__))
    
    return os.path.join(base_path, relative_path)
```

### 使用方式
```python
# 开发环境和打包后都能正确读取
txt_file_path = get_resource_path("售前通讯录.txt")
```
