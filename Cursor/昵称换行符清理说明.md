# 昵称换行符清理功能说明

## 修改日期
2026-01-26

## 修改目标
在数据加载过程中，自动清理**对象昵称**和**来源昵称**字段中的换行符，确保数据显示在单行内。

## 问题背景
部分数据源（数据库或文本文件）中的昵称字段可能包含换行符（`\n`、`\r`、`\r\n`），导致界面显示异常或数据导出格式错误。

## 解决方案

### 实现方式
在数据组装阶段统一处理，对所有昵称字段应用换行符清理函数。

### 处理规则
- 将 `\r\n`、`\n`、`\r` 替换为空格
- 保留多余空格（不做trim处理）
- 仅处理昵称字段，其他字段不受影响

## 修改内容

### 1. 新增辅助函数（第139-160行）

```python
def clean_newlines(text: str) -> str:
    """
    去除文本中的换行符，用空格替换
    
    功能：
        - 将 \r\n、\n、\r 替换为空格
        - 保留多余空格（不做trim处理）
        - 用于清理昵称字段中的换行符
    
    Args:
        text: 待清理的文本
    
    Returns:
        清理后的文本（换行符已替换为空格）
    
    示例：
        "张三\n李四" -> "张三 李四"
        "王五\r\n赵六" -> "王五 赵六"
    """
    if not text:
        return ""
    # 按顺序替换：先处理 \r\n（避免被拆分），再处理单独的 \n 和 \r
    return text.replace('\r\n', ' ').replace('\n', ' ').replace('\r', ' ')
```
